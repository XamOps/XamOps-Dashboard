Follow the official AWS guide to install the OTEL agent. Read manual

Installing the AWS OTEL agent may result in additional CloudWatch costs.

1. Ensure your command line is configured with credentials for your AWS account.

2. Run the official AWS command to deploy the Container Insights agent:

curl https://raw.githubusercontent.com/aws-observability/aws-otel-collector/main/deployment-template/eks/otel-container-insights-infra.yaml | kubectl apply -f -

After a successful installation, it may take up to 15 minutes for performance metrics to appear.
